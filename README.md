# Curso Programador Web - EaD 2026

## Objetivo do Repositório
Este repositório foi criado para armazenar e organizar todos os estudos, exercícios e projetos desenvolvidos durante o curso de Programador Web.

## O que o projeto contém
* **Documentação:** Explicações sobre lógica de programação, variáveis e repositórios remotos.
* **Exercícios:** Pastas com atividades práticas de lógica e frontend.
* **Materiais de Apoio:** PDFs e prints das aulas para consulta rápida.

## Como visualizar os arquivos
Os arquivos estão organizados em pastas. Para ler as explicações, basta navegar até a pasta `docs` e clicar nos arquivos `.md`. O conteúdo será renderizado diretamente pelo navegador.
