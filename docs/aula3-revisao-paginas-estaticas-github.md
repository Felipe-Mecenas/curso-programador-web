# Páginas Estáticas no GitHub (GitHub Pages)

O GitHub Pages é um serviço de hospedagem que permite transformar um repositório em um site estático.

## O que são Páginas Estáticas?
São sites compostos por arquivos que não mudam (HTML, CSS e JS) e não dependem de um servidor de banco de dados para processar o conteúdo.

## Como funciona no GitHub:
1. **Ativação**: Pode ser ativado nas configurações do repositório (Settings > Pages).
2. **Fonte**: O site pode ser gerado a partir da branch `main` ou de uma pasta específica (geralmente a pasta `docs`).
3. **Endereço**: O GitHub fornece uma URL pública no formato `seu-usuario.github.io/nome-do-repositorio`.

## Vantagens:
* Hospedagem gratuita.
* Integração direta com o versionamento Git.
* Excelente para portfólios e documentação de projetos.
