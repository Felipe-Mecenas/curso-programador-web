# Estruturas de Decisão

As estruturas de decisão permitem que um programa execute diferentes blocos de código com base em condições lógicas.

## 1. Estrutura Simples (Se / If)
Verifica uma condição. Se for verdadeira, executa o comando.

## 2. Estrutura Composta (Se-Senão / If-Else)
Oferece um caminho alternativo caso a condição inicial seja falsa.

## 3. Estrutura Encadeada (If-Else If-Else)
Utilizada para verificar múltiplas condições em sequência.

## Exemplo em Pseudocódigo:
SE idade >= 18 ENTÃO
    Exibir "Maior de idade"
SENÃO
    Exibir "Menor de idade"
FIM-SE
