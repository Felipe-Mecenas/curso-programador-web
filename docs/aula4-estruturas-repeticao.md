# Estruturas de Repetição

As estruturas de repetição (ou laços) permitem que um bloco de código seja executado várias vezes enquanto uma condição for verdadeira.

## 1. Estrutura Enquanto (While)
Repete o código enquanto a condição inicial for verdadeira. A verificação ocorre antes de cada execução.

## 2. Estrutura Para (For)
Utilizada quando sabemos antecipadamente o número de repetições. É ideal para percorrer listas ou coleções de dados.

## 3. Estrutura Faça-Enquanto (Do-While)
Garante que o código seja executado pelo menos uma vez antes de verificar a condição.

## Exemplo Conceitual:
PARA i DE 1 ATÉ 10 FAÇA
    EXIBIR "Contando: " + i
FIM-PARA
